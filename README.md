Single source of truth for the Ganaderia Catorce logo and brand mark.

## Logo Structure

The complete logo is one single object containing all four elements:

```
     GANADERIA
  [cow]    [pig]
     CATORCE
     EST. 2024
```

- **Cow + Pig**: Red line-art heads facing each other
- **GANADERIA**: Arched text above the icons
- **CATORCE**: Bold text below the icons
- **EST. 2024**: Establishment year

## Variants

| Variant | Use Case | Background |
|---------|----------|------------|
| `logo.png` | Default — dark text, red icons | Light backgrounds |
| `logo_light.png` | Light version — white text, red icons | Dark backgrounds |
| `logo_watermark.png` | Faded/muted version | Overlays, subtle branding |

## Brand Colors

- **Red**: `#D4231A`
- **Black**: `#0A0A0A`
- **White**: `#F5F5F5`

## Fonts Used in Logo

- **GANADERIA**: Josefin Sans 300
- **CATORCE**: Orbitron 700
- **EST. 2024**: Inter 400

## Usage

Copy the appropriate variant into your project's `assets/` folder. Size with CSS — the logo is a single self-contained graphic at each variant.
