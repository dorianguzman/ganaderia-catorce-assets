Single source of truth for the Ganaderia Catorce logo and brand mark.

## Logo Structure

The complete logo is one single object containing all four elements:

```
     GANADERIA
  [cow]    [pig]
     CATORCE
     EST. 2024
```

- **Cow + Pig**: Red line-art heads facing each other
- **GANADERIA**: Josefin Sans 300 italic
- **CATORCE**: Orbitron 800
- **EST. 2024**: Inter 400

## Brand Colors

- **Rojo Catorce**: `#D4231A`
- **Negro Humo**: `#0A0A0A`
- **Blanco Hueso**: `#F5F5F5`

## Usage

Copy the appropriate variant into your project's `assets/` folder. Size with CSS — the logo is a single self-contained graphic at each variant.

For the complete brand identity guide (colors, typography, photography direction, design principles), see the [Identidad de Marca](https://dorianguzman.github.io/ganaderia-catorce-assets/).
